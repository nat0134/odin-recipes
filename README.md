# odin-recipes

A basic HTML project for The Odin Project containing multi-page recipe website.

## Live Demo

https://github.com/nat0134/odin-recipes#

## About The Project

This project demonstrates basic front-end web developement skiils using only HTML. The website consists of a main index page which provide links to a number of recipes.

It presents an understanding and use of:
- Semantic HTML structure
- Relative links
- Lists and images
- Folder organization

## Built With

- HTML5
- Git
- GitHub
